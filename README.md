# Antigravity Skill Ecosystem 🌌

Bienvenido al Ecosistema de Skills de Antigravity. Este "Cerebro" técnico y personal está diseñado bajo una estricta arquitectura orientada a grafos, donde un **CTO Orquestador** central enlaza módulos altamente especializados para ejecutar proyectos complejos de principio a fin.

## 🏗️ Topología del Ecosistema

El ecosistema está construido en base a un **Director (Orquestador Raíz)** que delega el trabajo a módulos que dominan disciplinas tecnológicas y académicas separadas, sin mezclar nunca sus contextos.

### 👑 La Dirección (Raíz)
- `SKILL.md`: El **CTO de IA Orquestador**. Recibe solicitudes inmensas (como "construir una startup"), las descompone en decenas de tareas matemáticas, y delega automáticamente el trabajo a los sub-módulos capaces.
- `ADVANCED.md`: Documentación técnica avanzada para la IA que habilita la ejecución paralela profunda, emparejamiento de módulos por etiquetas y enrutamiento a prueba de fallos.

### 🧠 Enrutamiento Central
- **`orchestrating-workspace-brain`**: El "Cerebro" de tu espacio de trabajo. Recibe la carga diaria y decide estratégicamente hacia qué rama tecnológica derivar la tara. Mantiene el **Registro Maestro de Proyectos**.

### 🛠️ Los 4 Módulos de Operación Especializada
1. 🌐 **`engineering-web-microservices`**: Arquitectura web full-stack. Desarrolla React/Frontend clásico, microservicios Backend escalables, y conexiones rígidas de bases de datos relacionales en Supabase.
2. 🎨 **`engineering-modern-uis`**: Domina la Alta Ingeniería de Interfaces web (Puntaje de rendimiento máximo) usando los estándares de diseño y desempeño extremo para el 2026 (Server-Driven UI, Signals, OKLCH, aceleración GPU).
3. 🔌 **`developing-iot-systems`**: Laboratorio de hardware. Produce código C/C++ embebido extremadamente cuidado para evitar colapsar la memoria (Heap) de los ESP32, garantizando sistemas ininterrumpidos que mandan telemetría por MQTT.
4. ⚙️ **`automating-n8n-workflows`**: Operaciones Nerviosas. Construye arquitecturas de automatización en n8n impulsadas por Webhooks. Asegura ejecuciones "Idempotentes" impidiendo que las bases de datos de otros módulos recojan registros duplicados.

## 🔗 Estándares y Conexiones (Contracts & References)

La magia que permite a este software actuar de modo profesional recae en dos pilares internos de cada carpeta:
- **`contracts/` (Acuerdos):** Reglas JSON inquebrantables. Al comunicarse, el sensor de hardware, la interfaz web y la automatización se transmiten variables exactamente con la misma nomenclatura y estándar. 
- **`references/` (Mejores Prácticas):** Bases de conocimiento vitales específicas de ese nicho tecnológico para que la IA nunca cometa errores amateurs o "de novato" al escribir líneas de código para ti.

---

## 🚀 Cómo iniciar operaciones

1. Ordena a la Inteligencia Artificial: *"Quiero que el CTO orqueste un nuevo proyecto llamado [Nombre de Tu Idea]"*.
2. El sistema insertará y abrirá tú expediente en el **Registro Maestro** del Cerebro.
3. El plan será seccionado por el **CTO** y entregado equitativamente al FrontEnd, al BackEnd, o a la Placa Electrónica.
4. Disfruta un avance sistemático, trazable y pulcro.
